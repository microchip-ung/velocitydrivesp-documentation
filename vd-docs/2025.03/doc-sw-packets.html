SW Binary Packets

VelocityDRIVE-SP firmware is distributed as binary packets supporting a given Switch Chip family.

The binary packets follow the following naming schema:

    velocitydrivesp-<SWITCH-FAMILY>-bin-<VERSION>.tar.gz

Packets can be downloaded from here: http://mscc-ent-open-source.s3-website-eu-west-1.amazonaws.com/?prefix=public_root/velocitydrivesp/bin

The binary packets contain:

  • Full firmware images for the supported EVBs.

    • Located at: ./build-<SWITCH-FAMILY>/target/<SWITCH-FAMILY>/

    • A board already running VelocityDRIVE-SP firmware can be upgraded using the .fip file (see Firmware update for details).

    • Boards not running VelocityDRIVE-SP already (newly manufactured, bricked, or re-purposed) can be provisioned for VelocityDRIVE-SP using the .img files. See the de-brick procedure in the EVB section.

  • Raw executables ready for SW-Tailoring.

    • See SW Tailoring for details on the tailoring options.
