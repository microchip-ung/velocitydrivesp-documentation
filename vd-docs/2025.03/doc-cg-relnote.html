VelocityDRIVE-SP-2025.03 Release Note

Version:            2025.03
Previous Version:   2024.12
Release date:       2025-03-28
Type:               Roadmap release
YANG Catalog SHA:   440057a11e66eed82bc8e838347f694c

1. Announcement: First Public Release of VelocityDRIVE-SP

This is the first public release of VelocityDRIVE-SP, which includes the documentation and download of binaries.

This is not the first release; it is just the first public release. The release continues in its usual form and presents major changes since the last release.

2. New Features

  1. ACL (Access Control Lists): A new YANG to configure ACLs has been defined and implemented (details here).

  2. IETF-HW YANG (RFC 8348): The IETF Hardware YANG has been implemented (details here).

  3. 1588: Support for transparent clocks has been added (details here).

  4. 1588/802.1AS: L2 MAC and VLAN configuration is now supported (details here).

  5. L2: New attribute to disable forwarding to and from a given port (details here).

  6. VLAN: New attribute to control discard frames based on the inner tag (details here).

  7. VLAN: New attribute to specify a custom VLAN EtherType (details here).

  8. QoS: New feature to configure port policing added (details here).

  9. QoS: New feature to configure Weighted Round Robin scheduling on a port (details here).
